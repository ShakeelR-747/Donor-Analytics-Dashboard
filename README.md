# Donor Analytics Dashboard

## Project Overview

This project analyzes donor information using Power BI. It provides interactive dashboards to monitor donations, donor counts, payment methods, countries, and states.

## Tools Used

- Power BI
- Python
- Excel
- GitHub

## Features

- Total Donation KPI
- Total Donors
- New vs Existing Donors
- Donation by Month
- Donation by State
- Donation by Country
- Donation by Payment Mode
- Interactive Filters
